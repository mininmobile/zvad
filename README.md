# zvad
experimental minimalistic oligiosynthesic conlang

## alphabet
the zvad alphabet represented in latin script, [zvad's own writing system], and the ipa sounds of each letter.

| latin | zvad    | ipa |
| ----- | ------- | --- |
| z     | ട       | /ʒ/ |
| v     | ᑌ       | /β/ |
| b     | ᑐ       | /b/ |
| d     | ᑕ       | /d/ |
| a     | ⵔ       | /ɐ/ |
| e     | ⅄ or △  | /e/ |
